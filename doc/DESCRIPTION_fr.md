Ahwa réunit un petit « conseil » de personas IA pour délibérer sur
un dilemme que vous lui apportez. Chaque persona prend la parole à
tour de rôle autour d'une table virtuelle ; un synthétiseur prend
le pouls du conseil et en tire une perspective unifiée.

Conçu pour penser à voix haute, pas pour discuter. Utilisez-le pour
les arbitrages éthiques, les décisions de conception, les choix de
vie, l'examen critique de vos propres arguments — tout ce où une
réponse LLM en un seul coup aplatirait la nuance.

**Vie privée par conception.** Auto-hébergé, mono-utilisateur par
défaut, pas d'analytique, pas de télémétrie, pas d'appel sortant.
Vos dilemmes et les réponses du conseil vivent dans un fichier
SQLite local dans votre répertoire de données et ne vont nulle part
ailleurs.

**Apportez vos propres modèles.** Vercel AI SDK sous le capot ;
configurez n'importe quelle combinaison d'Anthropic, OpenAI,
OpenRouter, ou une instance Ollama locale.

**Open source.** AGPL-3.0-or-later.
